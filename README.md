# Disaster Relief Find

## Project Description
This project helps manage disaster relief resources using:
- Backend: Node.js + Express
- Frontend: React
- AI Server: Python + Flask
- Database: MongoDB

## Folder Structure
- backend/   → Node.js APIs
- frontend/  → React UI
- ai-server/ → Python AI logic

## How to Run the Project

### 1. Start MongoDB
Make sure MongoDB service is running.

### 2. Run Backend
```bash
cd backend
npx nodemon index.js
